# Hormuz-Shadow-Global-Shipping-Disruption-Analysis
Quantitative analysis of geopolitical disruption impacts on global maritime logistics using Python, statistical modeling, simulation analysis, and geospatial visualization.

## Project Overview

This research project analyzes the operational and economic impact of geopolitical disruption in the Strait of Hormuz using simulation modeling, statistical analysis, and geospatial visualization.

The study compares normal shipping routes with rerouted maritime paths around Africa to estimate increases in:

* Shipping distance
* Travel time
* Fuel cost
* Supply chain disruption risk

## Key Findings

* Rerouted shipping routes significantly increased fuel cost and travel time.
* Statistical testing confirmed significant operational differences.
* Regression analysis showed a strong positive relationship between shipping distance and fuel cost.
* Geospatial mapping visualized large-scale route deviations.
